# First Steps

## 1. Create Admin Account

On first launch, register an account. The first registered user automatically becomes admin.

## 2. Set Up Household

After login, a default household "My Household" is created. You can rename it under **Settings → Household**.

## 3. Add Accounts

Under **Accounts**, add your bank accounts with a starting balance.

## 4. Set Up Categories

Under **Categories**, configure income and expense categories. Set expected values for fixed costs and income.

## 5. Record Transactions

Under **Transactions**, record your income and expenses. You can also import CSV, OFX, and Excel files.
