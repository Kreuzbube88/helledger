# 🎯 Savings Goals

Under **Goals**, create savings goals and track their progress.

---

## Creating a Goal

| Field | Description |
|-------|-------------|
| Name | Goal label (e.g. "Vacation 2026") |
| Target Amount | Total amount to save |
| Target Date | When the goal should be reached (optional) |
| Linked Account | Savings account whose current balance counts as progress |
| Color | Color for the progress bar |
| Notes | Free-text notes |

---

## Progress

Progress is calculated automatically from the **current balance of the linked account**. Once the balance reaches the target amount, the goal is marked as **achieved**.

---

## Dashboard Widget

The three savings goals with the highest progress are shown as a widget on the **Dashboard** — with a progress bar and current / target amount.
