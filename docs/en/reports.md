# 📊 Reports & Overview

HELLEDGER provides several views for analyzing your finances.

---

## Dashboard

The dashboard gives a quick monthly overview:

- **Income / Expenses / Savings** — animated monthly totals
- **Available** — current balance split by account role (Fixed Costs / Variable / Savings)
- **KPIs** — savings rate, debt-to-income ratio, reserve coverage (months)
- **Expense Donut** — expenses by category
- **Reserve** — monthly reserve requirement for non-monthly fixed costs
- **Account Balances** — all accounts with current balance
- **Savings Goals** — top 3 goals by progress

---

## Month View

Accessible via **Month** in the navigation. Shows the selected month in detail:

- **KPI bar** — Income | Expenses | Savings (with rate) | Available
- **Income section** — all income by category
- **Fixed Costs section** — all fixed costs by category
- **Variable Expenses section** — all variable expenses by category
- **Savings Transfers section** — all savings transfers for the month

---

## Year View

Accessible via **Year** in the navigation. Shows a full category × month grid:

- All categories as rows
- 12 months as columns + **Total** column
- Planned values (from fixed costs) in italic with `~` prefix
- Savings transfers as a separate row

---

## Reports

Accessible via **Reports** in the navigation. Time period selectable (month / year / custom).

**Charts:**

| Chart | Description |
|-------|-------------|
| Expenses by Category | Pie chart of all expenses |
| Income & Expenses | Stacked bar chart (fixed + variable + savings) |
| Balance History | Line chart of account balance over time |
| Savings | Savings development over time |

---

## Forecast

Accessible via **Forecast** in the navigation. Shows a **12-month projection** based on active fixed costs:

- Monthly income, fixed costs, and savings transfers
- **Total Saved** — cumulative savings account balance month by month
- Line chart with all data points

> The forecast starts from the **next month**, since the current month is already booked with real transactions.
