# Import & Export

## CSV-Import

Unter **Transaktionen → Importieren** kannst du CSV-Dateien importieren. Format: `date,description,amount,type`

## OFX-Import

OFX-Dateien von Banken werden automatisch erkannt.

## Excel-Import

Excel-Dateien (.xlsx) können über **Transaktionen → Excel importieren** importiert werden. Spalten: date, description, amount, type.

## Excel-Export

Unter **Transaktionen → Excel exportieren** kannst du alle Transaktionen eines Jahres als Excel-Datei herunterladen.
