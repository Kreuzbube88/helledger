# Adminbereich

## Benutzerverwaltung

Unter **/admin** können Admins Benutzer sperren, entsperren und löschen.

## Systemstatus

Der Systemstatus zeigt aktuelle Metriken: Benutzeranzahl, Haushalte, Transaktionen und Datenbankgröße.

## Einstellungen

Im Admin-Bereich können SMTP-Einstellungen, OIDC-Konfiguration und Registrierungsfreigabe verwaltet werden.
