# Backup

HELLEDGER erstellt automatisch Backups der Datenbank im konfigurierten `BACKUP_PATH`.

## Manuelles Backup

Unter **Einstellungen → Backup** kannst du ein manuelles Backup erstellen und herunterladen.

## Backup-Interval

Standardmäßig wird alle 24 Stunden ein Backup erstellt. Konfigurierbar via `BACKUP_INTERVAL_HOURS`.
