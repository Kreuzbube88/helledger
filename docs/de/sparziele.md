# 🎯 Sparziele

Unter **Ziele** kannst du Sparziele anlegen und ihren Fortschritt verfolgen.

---

## Ziel anlegen

| Feld | Beschreibung |
|------|--------------|
| Name | Bezeichnung des Ziels (z. B. "Urlaub 2026") |
| Zielbetrag | Zu sparender Gesamtbetrag |
| Zieldatum | Wann das Ziel erreicht sein soll (optional) |
| Verknüpftes Konto | Sparkonto, dessen aktueller Saldo als Fortschritt gilt |
| Farbe | Farbe für den Fortschrittsbalken |
| Notizen | Freitext-Anmerkungen |

---

## Fortschritt

Der Fortschritt wird automatisch aus dem **aktuellen Saldo des verknüpften Kontos** berechnet. Sobald der Saldo den Zielbetrag erreicht, gilt das Ziel als **erreicht**.

---

## Dashboard-Widget

Die drei Sparziele mit dem höchsten Fortschritt werden im **Dashboard** als Widget angezeigt — mit Fortschrittsbalken und aktuellem / Ziel-Betrag.
